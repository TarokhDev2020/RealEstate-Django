
# RealEstate-Django

Real estate listings with an admin panel


## Packages

- django
- psycopg2-bindary
- pillow
- pylint-django


## Installation

Make sure to install pip for your virtual environment

```bash
  git clone https://github.com/TarokhDev2020/RealEstate-Django.git
  cd my-project
  pip install
```
